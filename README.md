# Weather App (Python CLI)

A simple command-line weather application built in Python that retrieves real-time weather data using the OpenWeatherMap API.

## Features
- Enter any city to retrieve current weather information
- Displays temperature in Celsius
- Shows humidity levels
- Provides weather condition description
- Uses real-time API data

## Technologies Used
- Python
- Requests library
- OpenWeatherMap API
- JSON data handling
- HTTP requests

## How It Works
The application sends a request to the OpenWeatherMap API using a user-provided city name. The response is returned in JSON format and parsed to extract relevant weather details such as temperature, humidity, and weather conditions.
